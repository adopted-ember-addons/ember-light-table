tableActions

Object

Defined in addon/components/light-table.js:43

This is used to propate custom user defined actions to custom cell and header components. As an example, lets say I have a table with a column defined with cellComponent: 'delete-user'

{{#light-table table tableActions=(hash
  deleteUser=(action 'deleteUser')
 ) as |t|}}
  {{t.head}}
  {{t.body}}
  {{t.foot}}
{{/light-table}}

Now in the delete-user component, we can access that deleteUser action and pass it the row object which will bubble all the way to where you defined that action.

<button {{action tableActions.deleteUser row}}>Delete Me</button>
